# Roadmap Co Website

A modern booking and company introduction website built with Next.js and Tailwind CSS.

## Features

- **Home Page**: Company introduction and hero section.
- **Booking System**: Mock booking form with success feedback.
- **Authentication**: Mock Sign In and Register functionality using Context API.
- **Modern UI**: Fully responsive design with Tailwind CSS.

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd roadmap-website
   ```

2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
